Simple ai wrapper.
usage: 
    gipwrap 
    -a | what ai to use, default chatgpt options: chatgpt, ollama
    -i | input file, default none and will take pipe/stdin
    -o | output file, default none and will print
    -s | system prompt file
    -S | system prompt
    -m | model to use
    -k | auth key os variable.
    -K | auth key raw.
    -A | enable agent mode for tool calling loops.
    -T | print intermediate agent thinking to stderr.
